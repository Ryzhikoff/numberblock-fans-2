# 🎬 Scene_69 — Пирамида Чисел: Быстрый Старт

## 🚀 За 3 шага

### Шаг 1: Открыть сцену
```
В Unity: File → Open Scene → Assets/Scenes/Scene_69.unity
```

### Шаг 2: Автоматическая настройка
```
В меню: Tools → Scene_69 → Quick Setup (One Click)
```

### Шаг 3: Запустить
```
Нажать Play ▶️
```

**Всё!** Через 2 секунды начнётся построение пирамиды чисел.

## 🛠️ Если что-то пошло не так

### Префабы не найдены?
Проверьте, что в `Assets/Prefabs/Blocks/` есть префабы с компонентом **Scale**:
- `1.prefab`, `2.prefab` ... `9.prefab`
- `10.prefab`, `20.prefab` ... `90.prefab`
- `100.prefab`, `200.prefab` ... `900.prefab`
- `1000.prefab`, `2000.prefab` ... `9000.prefab`
- `OneHundredThousand.prefab`
- `OneMillion.prefab`

### Ошибка в консоли?
1. Откройте консоль (Window → General → Console)
2. Проверьте, что все скрипты в `Assets/Scripts/` без ошибок компиляции
3. Перезапустите Unity

### Камера не движется?
Проверьте, что на GameManager назначен компонент `S69_Main` и в нём заполнено поле `Main Camera`.
