# 🎬 Scene 70: От 1 до ТРИЛЛИОНА - Космическое Путешествие

## 📋 Обзор

Эта сцена создаёт эпичное космическое путешествие камеры через блоки чисел от 1 до 1 триллиона с плавным облётом камеры, звёздным небом и кинематографичными эффектами.

## 🎯 Созданные скрипты

### 1. **S70_TrillionBlocks.cs** - Генератор блоков
- Создаёт блоки чисел по степеням 10 (1, 10, 100, 1000... до триллиона)
- Располагает их по спирали в 3D пространстве
- Реализует LOD систему (3 уровня детализации)
- Автоматически масштабирует блоки в зависимости от числа

### 2. **S70_CameraFlyPath.cs** - Система камеры
- **Автоматический режим**: Камера летит по сгенерированной спиральной траектории
- **Ручной режим**: Управление мышью + WASD
- Плавные переходы между точками пути с использованием кривых
- Горячие клавиши:
  - `Space` - Вкл/Выкл автоматический полёт
  - `WASD` - Движение
  - `Мышь` - Поворот камеры
  - `Space/Shift` - Вверх/Вниз
  - `R` - Перезапуск сцены

### 3. **S70_NumberDisplay.cs** - UI отображение числа
- Показывает текущее число и его название
- Анимация появления с масштабированием
- Эффект пульсирующего свечения
- Автоматическая смена чисел для демонстрации

### 4. **S70_Starfield.cs** - Звёздное небо
- 5000+ звёзд с разными цветами и размерами
- 20 туманностей с полупрозрачными материалами
- 1000 частиц для атмосферы
- Медленное вращение для динамики

### 5. **S70_SceneManager.cs** - Менеджер сцены
- Объединяет все системы
- Настраивает освещение и окружение
- Управляет постобработкой
- Вспомогательные методы для форматирования чисел

## 🔧 Настройка сцены в Unity

### Шаг 1: Подготовка сцены

1. Откройте `Scene_70.unity`
2. Убедитесь, что есть:
   - Main Camera
   - Directional Light
   - Пустой GameObject для контейнера блоков

### Шаг 2: Добавление скриптов

1. **На Main Camera** добавьте:
   - `S70_CameraFlyPath.cs`

2. **Создайте пустые GameObject** и добавьте:
   - `BlocksContainer` → `S70_TrillionBlocks.cs`
   - `SceneManager` → `S70_SceneManager.cs`
   - `Starfield` → `S70_Starfield.cs`

3. **Создайте UI Canvas** и добавьте:
   - TextMeshProUGUI для числа
   - TextMeshProUGUI для названия
   - TextMeshProUGUI для описания
   - На UI объект добавьте `S70_NumberDisplay.cs`

### Шаг 3: Назначение префабов

В инспекторе `S70_TrillionBlocks` назначьте:

- **prefabOne** → Префаб блока "1" (например, `Assets/10.prefab` или свой)
- **prefabTen** → Префаб блока "10"
- **prefabHundred** → Префаб блока "100"
- **prefabThousand** → Префаб блока "1000"
- **prefabMillion** → `Assets/Prefabs/Blocks/OneMillion.prefab`
- **prefabBillion** → Можно использовать OneMillion с другим масштабом
- **prefabTrillion** → Можно использовать OneMillion с другим масштабом

### Шаг 4: Настройка UI

1. Создайте Canvas (если нет):
   - GameObject → UI → Canvas

2. Добавьте TextMeshPro объекты:
   - Правый клик на Canvas → UI → Text - TextMeshPro
   - Создайте 3 текстовых поля:
     - `NumberText` - для отображения числа (1,000,000)
     - `NameText` - для названия (Миллион)
     - `DescriptionText` - для описания

3. Назначите их в `S70_NumberDisplay`

### Шаг 5: Настройка камеры

В `S70_CameraFlyPath` на Main Camera:

- **flySpeed**: 1-5 (скорость полёта)
- **autoFly**: true (автоматический полёт)
- **autoGeneratePath**: true (автогенерация пути)
- **pathPointCount**: 50-100 (больше = плавнее)
- **pathRadius**: 200 (радиус спирали)
- **pathHeight**: 100 (высота пути)
- **pathSpirals**: 3 (количество витков)

### Шаг 6: Настройка блоков

В `S70_TrillionBlocks` на BlocksContainer:

- **spacing**: 2 (расстояние между блоками)
- **spiralRadius**: 50 (радиус спирали блоков)
- **spiralHeight**: 5 (высота витка)
- **blocksPerSpiral**: 100 (блоков на виток)
- **LOD Distances**: 
  - lodDistance1: 100
  - lodDistance2: 500
  - lodDistance3: 2000

### Шаг 7: Освещение и эффекты

В `S70_SceneManager`:

- **useCustomSkybox**: true
- **skyboxMaterial**: Назначьте материал скайбокса (например, из `Fantasy Skybox FREE`)
- **ambientLightColor**: Тёмно-синий (0.3, 0.3, 0.5)
- **lightIntensity**: 1.5

## 🐛 Решение проблем

**Блоки не появляются?**
- Проверьте, что префабы назначены
- Убедитесь, что container существует
- Проверьте консоль на ошибки

**Камера не летит?**
- Проверьте `autoFly = true`
- Убедитесь, что pathPoints сгенерированы
- Проверьте скорость (flySpeed > 0)

**UI не отображается?**
- Убедитесь, что Canvas создан правильно
- Проверьте, что TextMeshPro импортирован
- Назначьте текстовые поля в инспекторе

**Низкий FPS?**
- Уменьшите `starCount` в Starfield
- Уменьшите `pathPointCount`
- Включите LOD (уменьшите расстояния LOD)
- Отключите тени у дальних блоков
