# 🎬 Scene_68 — Полная автоматическая настройка

## 🚀 Быстрый старт (3 команды)

### 1. Открыть сцену
```
File → Open Scene → Assets/Scenes/Scene_68.unity
```

### 2. Создать префабы
```
Tools → S68 → Create Block Prefabs
```

**Что создаётся:**
- 10 префабов Древней эпохи (1-10)
- 10 префабов Средневековья (10-100)
- 10 префабов Индустриальной (100-1000)
- 6 префабов Современности (1000-Million)
- 4 префаба Будущего (Billion-Trillion)
- 3 префаба Вечности (Centillion+)

**Путь:** `Assets/Prefabs/Blocks/Scene68/`

### 3. Настроить сцену
```
Tools → Setup Scene 68
```

**Что настраивается:**
- ✅ Компоненты S68_Main, S68_Era, S68_Narrator
- ✅ Ссылки на камеру, UI, точки спавна
- ✅ 6 конфигураций эпох
- ✅ Повествование для каждой эпохи

### 4. Назначить префабы
```
Tools → S68 → Auto Assign Prefabs to Scene_68
```

**Что назначается:**
- ✅ Все префабы распределяются по эпохам
- ✅ Списки чисел заполняются автоматически
- ✅ Не нужно ничего перетаскивать вручную!

## ▶️ Готово! Нажимай Play

```
Нажми Play ▶
```

**Что произойдёт:**

## 📊 Меню инструментов Scene_68

Все команды доступны в меню **Tools → S68**:

| Команда | Описание |
|---------|----------|
| **Create Block Prefabs** | Создать префабы блоков |
| **Auto Assign Prefabs to Scene_68** | Автоматически назначить префабы |
| **Setup Scene 68** | Настроить сцену полностью |
| **Add Main Camera If Missing** | Добавить камеру если её нет |

### Debug меню (Tools → S68 → Debug)

| Команда | Описание |
|---------|----------|
| **Show Era Configs** | Показать конфигурации эпох в Console |

## ⚠️ Возможные проблемы

### Ошибка "Папка с префабами не найдена"
**Решение:** Сначала запусти `Tools → S68 → Create Block Prefabs`

### Ошибка "S68_Main не найден"
**Решение:** 
1. Открой `Scene_68.unity`
2. Запусти `Tools → Setup Scene 68`

### Префабы не назначаются
**Решение:**
1. Проверь Console на ошибки
2. Убедись, что префабы созданы
3. Запусти `Tools → S68 → Debug → Show Era Configs`

### Камеры нет на сцене
**Решение:** Запусти `Tools → S68 → Add Main Camera If Missing`

## 📝 Примечания

- Все операции безопасны — можно запускать повторно
- Префабы не перезаписываются если уже существуют
- Конфигурации эпох не сбрасываются при повторном запуске
- Для сброса — удали `Assets/Prefabs/Blocks/Scene68/` и запусти заново
